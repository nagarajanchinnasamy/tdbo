Proc/Method Detail
public method open {location}
This method needs to be called only once in an application to initialize the logging facility to log messages in location. If the file already exists, it is opened to append the log messages. Otherwise, the file is created.
Parameters:
location
Pathname of the file where messages are to be logged in
public method init {service {level warn} {slaveinterp ""}}
This method needs to be called only once per service in a class or namespace. It invokes logger::init command for the service. See TclLib's logger package documentation for more details.
Parameters:
service
Name of the service. Typically, this is name of the class or namespace.
level
Log level of the service.
slaveinterp
Slave interpreter for which the alias for log methods are created so that the log methods can be invoked from the slave interpreters.
Returns:
Logging command object, using which the logging methods can be invoked.
